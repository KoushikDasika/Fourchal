h1. Fourchal

h2. Ruby on Rails

This application requires:

* Ruby version 1.9.3
* Rails version 3.2.11

h2. Database

This application uses SQLite with ActiveRecord.

h2. Development

* Template Engine: Haml
* Testing Framework: RSpec and Factory Girl and Cucumber
* Front-end Framework: Twitter Bootstrap (Sass)
* Form Builder: None
* Authentication: Devise
* Authorization: CanCan

h2. Email

The application is configured to send email using a SendGrid account.

h2. Deployment
Try a live version of this project on <a href="www.fourchal.com">Heroku</a>

h2. Future Versions
This project is in the middle of a complete rewrite.  Keep an eye out for it
